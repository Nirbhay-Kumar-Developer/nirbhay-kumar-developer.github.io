About TOS

TOS (Terminal Operating System) is a lightweight, pre-configured terminal environment specifically designed for Termux users. It provides a complete development environment with project management capabilities.

Key Benefits

  • 🚀 Pre-configured development environment
  • 📦 Integrated project management
  • ⚡ Lightweight and fast
  • 🎨 Customizable interface
  • 🔧 Built-in utilities
  • 📈 System optimization tools
  • 💾 Efficient storage usage

Installation

Prerequisites

  • Termux installed
  • Android 7.0+
  • At least 200MB free space

Installation Steps

git clone https://github.com/Nirbhay-Kumar-Developer/TOS.git
cd TOS
./setup.sh

Common Commands

Command Description
tos init Initialize new project
tos status Show system status
tos optimize Optimize system
tos help Display help

Configuration

Edit ~/.tos/config to customize your environment:

[environment]
theme=dark
language=en
editor=nano

[performance]
auto_optimize=true
cache_size=100M

Troubleshooting

Installation fails

Ensure you have sufficient storage and all prerequisites installed.

Commands not found

Run tos install-completions to register commands.
