Overview

Worklog is a command-line tool designed to help developers track their daily achievements, mistakes, and learnings. It promotes reflection and continuous improvement through systematic daily logging.

Features

  • 📝 Easy daily logging
  • ✅ Track achievements
  • ❌ Log mistakes for learning
  • 💡 Record daily learnings
  • 📊 View historical data
  • 🔍 Search and filter entries
  • 📈 Generate reports
  • ⚡ Lightweight and fast

Installation

git clone https://github.com/Nirbhay-Kumar-Developer/Worklog.git
cd Worklog
make install

Basic Usage

worklog log "Completed feature X"
worklog mistake "Fixed bug in module Y"
worklog learn "Discovered new optimization technique"

Available Commands

Command Description
worklog log Log an achievement
worklog mistake Record a mistake
worklog learn Record a learning
worklog today Show today's entries
worklog week Show this week's summary
worklog search Search entries

Usage Examples

Log Today's Work

worklog log "Refactored database layer"
worklog mistake "Introduced race condition in v1.2"
worklog learn "Understanding mutex patterns better"

View Daily Summary

worklog today

Search Past Entries

worklog search "database"
