Introduction

This is a template repository for quickly starting new projects with mkapk. It includes pre-configured build scripts, directory structure, and example code for Java, Kotlin, and C++.

Getting Started

Use This Template

  1. Click "Use this template" on GitHub
  2. Create your new repository
  3. Clone your new repository
  4. Run setup: ./setup.sh
